Overview

Experience Lab is an automated, end-to-end deployment accelerator with a friendly web-based UI that provides easy, fast deployments of Microsoft Energy Data Services with sample dataset(s) for learning, testing, demo, and training purposes. This enables business audience and domain practitioners who are not familiar with OSDUTM to be able to create fully loaded and configured Microsoft Energy Data Services deployments.

It is not intended to be a guide to Microsoft Energy Data Services or intended to deploy in production environments.


Experience Lab provides several integration capabilities for Microsoft Energy Data Services:

  • Obtain User Access Tokens
  • Execute API Calls using Swagger
  • Perform a CSV Ingestion
  • Manage users & groups
  • Perform a Core Functionality Validation Test
  • Ingest a Sample Data Set (TNO)
  • Visualize TNO data in Power BI
