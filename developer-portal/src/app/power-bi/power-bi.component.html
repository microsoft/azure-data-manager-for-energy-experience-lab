Power BI Visualization

Included in Experience Lab is a pre-configured Power BI connector for visualizing TNO data. The connector is open source, and can be found on Kadri Umay's GitHub. Currently, the Power BI connector only supports client-side Power BI reports - it does not support reports published to Power BI Online.

Prerequisites

  • You must be using a Windows PC
  • You have ingested Open Test Data (TNO) into your Microsoft Energy Data Services instance

Instructions

The following steps explain how to use Power BI with Microsoft Energy Data Services.

  1. Download and install Power BI Desktop
  2. Enable unsigned connectors
    1. Open Power BI Desktop
    2. Navigate to File > Options and Settings > Options > Security
    3. Under "Data Extensions", select "Allow any extension to load without validation or warning" (Read more about custom connectors)
    4. Click OK
  3. Download Power BI Connector, and save it to C:\Users\<profile>\Documents\Power BI Desktop\Custom Connectors. Create this folder if it doesn't exist.
  4. Download the sample Power BI Template Report from GitHub
  5. Open the report in Power BI Desktop
