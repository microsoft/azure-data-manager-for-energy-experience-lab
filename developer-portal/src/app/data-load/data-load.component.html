TNO Data Load

Test features in Microsoft Energy Data Services after loading the TNO data set.

Note: Manifests are automatically assigned a data.tno.viewers and data.tno.owners Access Control List (ACL). The first user created during Experience Lab's deployment will be assigned these ACLs. Additional users will need to be a member of these ACLs to query and manipulate TNO data. Users can be added to these ACLs from the User Management page.

Overview

  1. Go to the open-test-data Template Spec in your Experience Lab Resource Group
  2. Click Deploy
  3. Click Review + Create and start the deployment
  4. The data load will run from a container instance in the same resource group. You can monitor progress in its logs.
  5. Wait approx. 1 hour for data load to complete.
